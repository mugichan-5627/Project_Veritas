# CapIQ Export Guide

CapIQ files are proprietary and must not be committed.

Expected local folders:

- `data/capiq/public_comps/global_peers/`
- `data/capiq/precedent_transactions/global_transactions/`

Use sector-level global exports with entity name, ticker, market capitalization,
revenue, EBITDA, EV/revenue, EV/EBITDA, announced date, transaction value, and
transaction multiple fields.
